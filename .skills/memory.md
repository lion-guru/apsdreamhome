# Memory & Context Skills

## Remember Project State
Use memory MCP to remember:
- Current task in progress
- Recently modified files
- Known bugs/issues
- User preferences

## Key Project Facts
- PHP MVC Framework (not Laravel)
- Custom routing system
- 100+ Controllers
- MySQL database on port 3307
- XAMPP stack

## Common Issues & Solutions
1. Header CSS not loading → Check base.php link tags
2. Route 404 → Check routes/web.php
3. DB error → Check .env credentials
4. Port conflict → Check netstat for port 80

## MCP Tools Quick Reference
- `filesystem`: grep, read with offset/limit
- `mysql`: Direct SQL queries
- `playwright`: Browser screenshot
- `memory`: Store/retrieve context
- `git`: Check history, diffs

## Token Saving Tips
1. Use grep before reading files
2. Read specific lines only (offset/limit)
3. Be concise in responses
4. Use MCP tools instead of file reads
