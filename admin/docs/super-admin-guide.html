सुपर एडमिन यूजर गाइड

1. टेम्पलेट मैनेजमेंट

नया टेम्पलेट बनाना:

  1. टेम्पलेट लिस्ट पेज पर जाएं
  2. "Create New Template" बटन पर क्लिक करें
  3. टेम्पलेट का नाम और विवरण दर्ज करें
  4. कंटेंट एडिटर में टेम्पलेट का डिज़ाइन करें
  5. Save बटन पर क्लिक करें

टेम्पलेट एडिट करना:

  1. टेम्पलेट लिस्ट में से टेम्पलेट चुनें
  2. Edit बटन पर क्लिक करें
  3. आवश्यक बदलाव करें
  4. Save बटन पर क्लिक करें

2. यूजर मैनेजमेंट

नया यूजर बनाना:

  1. एडमिन लिस्ट पेज पर जाएं
  2. "Add New Admin" बटन पर क्लिक करें
  3. यूजर की जानकारी भरें
  4. रोल और परमिशन सेट करें
  5. Save बटन पर क्लिक करें

यूजर परमिशन मैनेज करना:

  1. एडमिन लिस्ट में से यूजर चुनें
  2. Edit बटन पर क्लिक करें
  3. परमिशन सेक्शन में जाएं
  4. आवश्यक परमिशन अपडेट करें
  5. Save बटन पर क्लिक करें

3. कंटेंट मैनेजमेंट

पेज कंटेंट अपडेट करना:

  1. कंटेंट मैनेजमेंट सेक्शन में जाएं
  2. एडिट करने के लिए पेज चुनें
  3. कंटेंट एडिटर में बदलाव करें
  4. प्रीव्यू चेक करें
  5. Publish बटन पर क्लिक करें

4. महत्वपूर्ण टिप्स

सुरक्षा टिप: हमेशा मजबूत पासवर्ड का उपयोग करें और नियमित रूप से पासवर्ड बदलें।

बैकअप टिप: महत्वपूर्ण बदलाव करने से पहले बैकअप लें।

परमिशन टिप: यूजर्स को केवल वही परमिशन दें जो उन्हें वास्तव में चाहिए।
