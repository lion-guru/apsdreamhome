# Frequently Asked Questions (FAQ)

## General

### What is APS Dream Home?
APS Dream Home is a comprehensive real estate management system designed to help real estate agencies manage properties, clients, and transactions efficiently.

### What are the system requirements?
- PHP 7.4 or higher
- MySQL 5.7+ or MariaDB 10.3+
- Web server (Apache/Nginx)
- Modern web browser

## Installation

### How do I install APS Dream Home?
Please refer to our [Getting Started](/docs/?doc=getting-started) guide for detailed installation instructions.

### I'm getting a database connection error. What should I do?
1. Verify your database credentials in the `.env` file
2. Ensure the database server is running
3. Check if the database user has proper permissions

## User Management

### How do I reset a user's password?
1. Go to Admin Panel > Users
2. Find the user and click 'Edit'
3. Click 'Reset Password' and follow the prompts

### What's the difference between user roles?
- **Admin**: Full system access
- **Agent**: Can manage properties and clients
- **User**: Basic access to view properties and submit inquiries

## Property Management

### How do I add a new property?
1. Navigate to Properties > Add New
2. Fill in all required fields
3. Upload property images
4. Click 'Publish'

### Can I import properties in bulk?
Yes, you can use our CSV import feature under Properties > Import.

## Troubleshooting

### The dashboard is loading slowly. What can I do?
1. Clear your browser cache
2. Check server resources
3. Optimize your database
4. Contact your hosting provider

### I'm getting a 500 error. What should I do?
1. Check the error logs in `storage/logs`
2. Verify file permissions
3. Make sure all dependencies are installed

## Property Listings

### How do I list a property?
1. Log in to your account
2. Go to "My Properties"
3. Click "Add New Property"
4. Fill in the property details
5. Upload photos
6. Click "Publish"

### Can I edit or remove a property listing?
Yes, you can edit or remove your property listings at any time from your dashboard.

## Property Visits

### How do I schedule a property visit?
1. Find the property you're interested in
2. Click "Schedule a Visit"
3. Select an available date and time
4. Provide your contact information
5. Confirm the appointment

### Can I reschedule or cancel a visit?
Yes, you can reschedule or cancel visits through the confirmation email or your account dashboard.

## Lead Management

### How are leads assigned to agents?
Leads are automatically assigned to the agent who owns the property or can be manually assigned by an admin.

### How can I track my leads?
You can track all your leads in the "My Leads" section of your dashboard, where you can see status, notes, and follow-up actions.
