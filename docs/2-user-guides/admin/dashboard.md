# Admin Dashboard Guide

## Overview

The Admin Dashboard is the central control panel for managing the APS Dream Home system. This guide covers all the features and functionality available to administrators.

## Accessing the Dashboard

1. Navigate to `/admin`
2. Enter your admin credentials
3. Click "Login"

## Dashboard Layout

### 1. Navigation Sidebar
- **Dashboard**: Overview of system metrics
- **Properties**: Manage property listings
- **Leads**: View and manage leads
- **Visits**: Schedule and track property visits
- **Users**: Manage user accounts and permissions
- **Settings**: System configuration

### 2. Header
- Quick search
- Notifications
- User menu
- Help/Support

### 3. Main Content Area
- Overview widgets
- Recent activities
- Quick actions

## Key Features

### Property Management
- Add/Edit/Delete properties
- Manage property details and media
- Set property status (Available/Sold/Under Contract)
- Manage property features and amenities

### Lead Management
- View all leads
- Filter and sort leads
- Assign leads to agents
- Track lead status
- View lead history

### Visit Scheduling
- Schedule property visits
- Send visit confirmations
- Track visit status
- Manage visit reminders

### User Management
- Create and manage user accounts
- Set user roles and permissions
- Reset passwords
- Manage user sessions

### Reports
- Generate property reports
- View lead conversion metrics
- Track agent performance
- Export reports to PDF/Excel

## Quick Start Guide

### Adding a New Property
1. Click "Properties" in the sidebar
2. Click "Add New Property"
3. Fill in property details
4. Upload property images
5. Click "Save"

### Managing Leads
1. Go to "Leads"
2. Click on a lead to view details
3. Update lead status
4. Add notes or assign to an agent

### Scheduling a Visit
1. Navigate to "Visits"
2. Click "Schedule Visit"
3. Select property and customer
4. Choose date and time
5. Click "Confirm"

## Tips & Best Practices

1. **Regular Backups**: Always back up your data regularly
2. **User Permissions**: Assign appropriate permissions to users
3. **Lead Follow-up**: Follow up on leads within 24 hours
4. **Property Updates**: Keep property information up to date
5. **System Updates**: Keep the system updated to the latest version

## Troubleshooting

### Common Issues

#### Can't Login
- Check your username and password
- Ensure CAPS LOCK is off
- Try resetting your password

#### Slow Performance
- Clear your browser cache
- Check your internet connection
- Contact support if the issue persists

#### Missing Features
- Check your user permissions
- Ensure you're using a supported browser
- Clear your browser cache and refresh the page
