# User Management Guide

## Overview

This guide explains how to manage user accounts, roles, and permissions in the APS Dream Home system.

## Accessing User Management

1. Log in to the admin dashboard
2. Click on "Users" in the sidebar

## User List View

- **Search**: Search users by name, email, or username
- **Filters**: Filter by role, status, or registration date
- **Columns**:
  - Name
  - Email
  - Role
  - Status
  - Last Login
  - Actions

## Adding a New User

### Basic Information
1. Click "Add New User"
2. Fill in the form:
   - First Name
   - Last Name
   - Email Address
   - Username (auto-generated if left blank)
   - Password (auto-generated if left blank)
   - Phone Number
   - Profile Picture (optional)

### User Role Assignment
1. Select the appropriate role:
   - **Super Admin**: Full system access
   - **Admin**: Full access except system settings
   - **Agent**: Can manage properties and leads
   - **Customer**: Can view properties and submit inquiries

### Account Settings
1. **Account Status**:
   - Active: User can log in
   - Inactive: User cannot log in
   - Suspended: Temporary account suspension

2. **Email Notification**:
   - Send welcome email with login details
   - Require password change on first login

3. Click "Create User" to save

## Editing User Information

1. Find the user in the list
2. Click the "Edit" button
3. Update the necessary fields
4. Click "Update User" to save changes

## Managing User Roles

### Available Roles
- **Super Admin**: Full system access
- **Admin**: Manage users, properties, and content
- **Agent**: Manage assigned properties and leads
- **Customer**: Basic access to view properties

### Changing a User's Role
1. Edit the user
2. Select a new role from the dropdown
3. Click "Update User"

## User Permissions

### Viewing Permissions
1. Go to Users > Permissions
2. View all available permissions

### Managing Role Permissions
1. Go to Users > Roles
2. Click "Edit" next to a role
3. Check/uncheck permissions as needed
4. Click "Update Role"

## User Status Management

### Available Statuses
- **Active**: User can log in
- **Inactive**: User cannot log in
- **Suspended**: Temporary suspension with reason
- **Pending**: Awaiting email verification

### Changing User Status
1. Find the user in the list
2. Click the status dropdown
3. Select new status
4. Add a reason if suspending
5. Confirm the change

## Resetting User Passwords

1. Find the user in the list
2. Click the "Reset Password" button
3. Choose an option:
   - Send password reset link
   - Set new password directly
4. Confirm the action

## Bulk User Actions

### Available Actions
- Activate/Deactivate users
- Change user roles
- Delete users
- Export user list

### How to Use Bulk Actions
1. Select users using checkboxes
2. Choose an action from the dropdown
3. Click "Apply"
4. Confirm the action

## User Profile Management

### Viewing User Profile
1. Click on a user's name
2. View detailed profile information
3. See activity history

### Editing Your Own Profile
1. Click on your profile picture
2. Select "My Profile"
3. Update your information
4. Click "Update Profile"

## User Activity Logs

### Viewing Activity Logs
1. Go to Users > Activity Logs
2. Filter by user, date, or action
3. View detailed activity history

## User Registration Settings

### Managing Registration
1. Go to Settings > User Registration
2. Configure options:
   - Allow new registrations
   - Require email verification
   - Default user role
   - Registration form fields

## Two-Factor Authentication

### Enabling 2FA
1. Go to Users > Two-Factor Auth
2. Enable 2FA for specific roles
3. Configure 2FA methods:
   - Authenticator apps
   - SMS verification
   - Email verification

## User Import/Export

### Importing Users
1. Go to Users > Import
2. Download the sample CSV file
3. Fill in user data
4. Upload the file
5. Map the fields
6. Click "Import Users"

### Exporting Users
1. Go to Users > Export
2. Select users to export
3. Choose export format (CSV/Excel)
4. Select fields to include
5. Click "Export Users"

## User Sessions

### Viewing Active Sessions
1. Go to Users > Active Sessions
2. View all active user sessions
3. See device and location information

### Managing Sessions
1. View session details
2. Log out specific sessions
3. View login history

## User Groups

### Creating User Groups
1. Go to Users > Groups
2. Click "Add New Group"
3. Enter group name and description
4. Add users to the group
5. Set group permissions

### Managing Group Members
1. Edit the group
2. Add/remove users
3. Set group managers
4. Configure group settings

## User Communication

### Sending Messages
1. Go to Users > Messages
2. Click "New Message"
3. Select recipients
4. Compose your message
5. Add attachments if needed
6. Click "Send"

### Email Templates
1. Go to Settings > Email Templates
2. Edit default templates
3. Create custom templates
4. Preview before sending

## User Privacy

### Privacy Settings
1. Go to Settings > Privacy
2. Configure data retention policies
3. Set user data export options
4. Configure account deletion settings

## Troubleshooting

### Common Issues

#### User Cannot Log In
- Check account status
- Verify email and password
- Check if account is locked
- Verify email verification status

#### Password Reset Not Working
- Check spam/junk folder
- Verify email server settings
- Ensure user email is correct

#### Permission Issues
- Check user role assignments
- Verify role permissions
- Clear user cache
