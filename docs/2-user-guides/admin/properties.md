# Property Management Guide

## Overview

This guide explains how to manage properties in the APS Dream Home system, including adding, editing, and managing property listings.

## Accessing Property Management

1. Log in to the admin dashboard
2. Click on "Properties" in the sidebar

## Property List View

- **Search Bar**: Search properties by name, location, or ID
- **Filters**: Filter by status, type, price range, etc.
- **Sorting**: Sort by date, price, or name
- **Bulk Actions**: Select multiple properties for bulk operations

## Adding a New Property

### Basic Information
1. Click "Add New Property"
2. Fill in basic details:
   - Property Title
   - Property Type (House, Apartment, Land, etc.)
   - Price
   - Location
   - Status (Available, Sold, Under Contract)

### Detailed Information
1. **Description**: Add a detailed property description
2. **Features**: Select property features (e.g., Pool, Garden, Garage)
3. **Specifications**:
   - Bedrooms
   - Bathrooms
   - Area (sq ft/m²)
   - Year Built
   - Floor Level

### Media Management
1. **Images**: Upload high-quality property images
   - Recommended size: 1200x800px
   - Maximum 20 images per property
   - Set a featured image
2. **Videos**: Add property tour videos (YouTube/Vimeo links)
3. **Virtual Tour**: Add 360° virtual tour if available

### Location & Map
1. Enter the full address
2. Drag the map pin to set exact location
3. Add nearby landmarks (optional)

### Additional Options
1. **Featured Property**: Mark as featured
2. **Hot Offer**: Mark as a special offer
3. **Open House**: Schedule open house dates

### Saving the Property
1. Click "Save as Draft" to save without publishing
2. Click "Publish" to make the property live
3. Click "Preview" to see how it will appear to users

## Editing a Property

1. Find the property in the list
2. Click the "Edit" button
3. Make necessary changes
4. Click "Update" to save changes

## Managing Property Status

### Available Statuses
- **Available**: Property is available for purchase/rent
- **Under Contract**: Offer has been made
- **Sold/Rented**: Property is no longer available
- **Draft**: Not visible to public

### Changing Status
1. Go to the property list
2. Find the property
3. Click the status dropdown
4. Select new status
5. Confirm the change

## Property Features & Amenities

### Adding Custom Features
1. Go to Settings > Property Features
2. Click "Add New Feature"
3. Enter feature name and icon
4. Click "Save"

### Assigning Features to Properties
1. Edit a property
2. Go to the "Features" tab
3. Check the features that apply
4. Click "Update"

## Property Media Management

### Uploading Images
1. Edit the property
2. Go to the "Media" tab
3. Click "Upload Images"
4. Select files or drag and drop
5. Set featured image by clicking the star icon

### Reordering Images
1. Go to the "Media" tab
2. Drag and drop images to reorder
3. The first image will be the featured image

### Deleting Images
1. Go to the "Media" tab
2. Hover over an image
3. Click the trash icon
4. Confirm deletion

## Property Documents

### Adding Documents
1. Go to the "Documents" tab
2. Click "Add Document"
3. Select document type (PDF, DOC, etc.)
4. Upload the file
5. Add a title and description

### Managing Documents
- View all uploaded documents
- Download or delete documents
- Set document visibility (Public/Private)

## Property Analytics

### Viewing Analytics
1. Go to the property details
2. Click on the "Analytics" tab
3. View:
   - Total views
   - Lead conversions
   - Popular times
   - User engagement

## Bulk Actions

### Available Actions
- Publish multiple properties
- Move to draft
- Change status
- Delete properties
- Export to CSV/Excel

### How to Use Bulk Actions
1. Select properties using checkboxes
2. Choose an action from the dropdown
3. Click "Apply"
4. Confirm the action

## Import/Export Properties

### Importing Properties
1. Go to Properties > Import
2. Download the sample CSV file
3. Fill in your property data
4. Upload the file
5. Map the fields
6. Click "Import"

### Exporting Properties
1. Go to Properties > Export
2. Select properties to export
3. Choose export format (CSV/Excel)
4. Click "Export"

## Property Reviews

### Managing Reviews
1. Go to Properties > Reviews
2. View all property reviews
3. Approve or reject reviews
4. Respond to reviews

## Tips for Better Listings

1. **High-Quality Images**: Use professional photos
2. **Detailed Descriptions**: Be thorough and honest
3. **Accurate Pricing**: Keep prices up to date
4. **Quick Responses**: Respond to inquiries promptly
5. **Regular Updates**: Keep listings current

## Troubleshooting

### Common Issues

#### Images Not Uploading
- Check file size (max 5MB per image)
- Ensure file format is JPG, PNG, or GIF
- Check server upload limits

#### Property Not Showing
- Check if status is set to "Published"
- Verify publish date is not in the future
- Clear cache and refresh

#### Map Not Displaying
- Verify Google Maps API key is valid
- Check if location coordinates are correct
- Ensure JavaScript is enabled
