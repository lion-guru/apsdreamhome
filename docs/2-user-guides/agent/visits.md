# Property Visits Management Guide

## Overview

This guide explains how to manage property visits, schedule viewings, and track visit outcomes in the APS Dream Home system.

## Accessing Visits

1. Log in to your agent dashboard
2. Click on "Visits" in the sidebar

## Visit Calendar View

### Calendar Navigation
- **Views**: Day, Week, Month, Agenda
- **Navigation**: Previous/Next buttons, Today
- **Filtering**:
  - By Property
  - By Status
  - By Agent

### Color Coding
- **Scheduled**: Blue
- **Confirmed**: Green
- **Completed**: Gray
- **Cancelled**: Red
- **No-show**: Orange

## Scheduling a Visit

### Quick Add
1. Click "+ New Visit"
2. Fill in details:
   - Property
   - Lead/Customer
   - Date & Time
   - Duration
   - Visit Type (In-person, Virtual)
   - Notes
3. Click "Save"

### From Lead/Property
1. Open the lead/property
2. Click "Schedule Visit"
3. Fill in details
4. Click "Confirm"

## Managing Visits

### Viewing Visit Details
1. Click on a visit in the calendar
2. View/Edit:
   - Basic Info
   - Attendees
   - Notes
   - Status
   - Follow-up Tasks

### Updating Visit Status
1. Open the visit
2. Click "Update Status"
3. Select new status:
   - Scheduled
   - Confirmed
   - Completed
   - Cancelled
   - No-show
4. Add notes if needed
5. Click "Update"

### Rescheduling Visits
1. Open the visit
2. Click "Reschedule"
3. Select new date/time
4. Add reason (optional)
5. Click "Reschedule"

### Cancelling Visits
1. Open the visit
2. Click "Cancel Visit"
3. Select reason
4. Add notes
5. Click "Confirm Cancellation"

## Visit Confirmation

### Automated Confirmations
- Email/SMS sent 24h before
- Includes:
  - Property details
  - Date/Time
  - Location
  - Agent contact
  - Cancellation link

### Manual Confirmation
1. Open the visit
2. Click "Send Confirmation"
3. Choose channel (Email/SMS)
4. Customize message
5. Click "Send"

## Visit Follow-up

### Adding Notes
1. Open the visit
2. Go to "Notes" tab
3. Click "Add Note"
4. Enter details
5. Set note type (Call, Email, Feedback)
6. Click "Save"

### Creating Follow-up Tasks
1. Open the visit
2. Click "Add Follow-up"
3. Set due date
4. Enter task details
5. Assign to team member
6. Click "Create"

### Sending Thank You
1. Open the visit
2. Click "Send Thank You"
3. Choose template
4. Personalize message
5. Click "Send"

## Visit Reports

### Available Reports
- Visit Volume
- Conversion Rate
- Agent Performance
- Cancellation Reasons
- Lead Source Analysis

### Generating Reports
1. Go to Reports > Visits
2. Select report type
3. Set date range
4. Apply filters
5. Click "Generate"
6. Export as needed (PDF, Excel)

## Best Practices

### Before the Visit
- Confirm 24h in advance
- Review property details
- Prepare materials
- Check traffic/weather

### During the Visit
- Arrive early
- Follow safety protocols
- Take notes
- Address questions

### After the Visit
- Send thank you
- Log feedback
- Schedule follow-up
- Update CRM

## Troubleshooting

### Common Issues

#### Double Bookings
- Check calendar conflicts
- Set buffer times
- Enable availability sync

#### No-shows
- Send reminders
- Call to confirm
- Log reason
- Update lead status

#### Scheduling Conflicts
- Check agent availability
- Enable calendar sync
- Set working hours
