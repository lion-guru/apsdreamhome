# Agent Leads Management Guide

## Overview

This guide explains how to manage leads in the APS Dream Home system as an agent. Learn how to track, follow up, and convert leads into clients.

## Accessing Leads

1. Log in to your agent dashboard
2. Click on "Leads" in the sidebar

## Lead List View

- **Search**: Search leads by name, email, or phone
- **Filters**:
  - Status (New, Contacted, Qualified, etc.)
  - Source (Website, Phone, Referral, etc.)
  - Date Range
  - Assigned Agent
- **Columns**:
  - Lead Name
  - Contact Info
  - Status
  - Source
  - Last Contact
  - Actions

## Lead Statuses

### Default Statuses
- **New**: Recently created lead
- **Contacted**: Initial contact made
- **Qualified**: Lead meets criteria
- **Proposal Sent**: Proposal shared
- **Negotiation**: In discussions
- **Won**: Lead converted to client
- **Lost**: Not interested
- **Nurture**: Future potential

### Custom Statuses
1. Go to Settings > Lead Settings
2. Click "Add New Status"
3. Enter status name and color
4. Set as active/inactive

## Adding a New Lead

### Manual Entry
1. Click "Add New Lead"
2. Fill in contact details:
   - First Name
   - Last Name
   - Email
   - Phone
   - Preferred Contact Method
3. Add lead details:
   - Status
   - Source
   - Assigned To
   - Lead Score
   - Tags
4. Add notes or comments
5. Click "Save Lead"

### Importing Leads
1. Click "Import Leads"
2. Download the template
3. Fill in lead information
4. Upload the file
5. Map the fields
6. Click "Import"

## Managing Leads

### Viewing Lead Details
1. Click on a lead's name
2. View the lead's:
   - Contact Information
   - Interaction History
   - Notes
   - Associated Properties
   - Documents

### Updating Lead Status
1. Find the lead in the list
2. Click the status dropdown
3. Select new status
4. Add a note (optional)
5. Click "Update"

### Adding Notes
1. Open the lead details
2. Go to the "Notes" tab
3. Click "Add Note"
4. Enter your note
5. Set note type (Call, Email, Meeting, etc.)
6. Click "Save Note"

### Scheduling Follow-ups
1. Open the lead details
2. Click "Schedule Follow-up"
3. Select date and time
4. Set follow-up type
5. Add notes
6. Click "Schedule"

## Lead Communication

### Sending Emails
1. Open the lead
2. Click "Send Email"
3. Choose a template or write custom
4. Add attachments if needed
5. Click "Send"

### Logging Calls
1. Click "Log Call"
2. Enter call details:
   - Call Type (Inbound/Outbound)
   - Duration
   - Summary
   - Next Steps
3. Click "Save"

### Text Messaging
1. Click "Send SMS"
2. Type your message
3. Preview
4. Click "Send"

## Lead Assignment

### Assigning to Yourself
1. Find the lead
2. Click "Assign to Me"
3. Confirm assignment

### Reassigning Leads
1. Select the lead(s)
2. Click "Reassign"
3. Select new agent
4. Add a note (optional)
5. Click "Reassign"

## Lead Scoring

### How It Works
- Points assigned based on actions
- Higher score = hotter lead
- Automated scoring rules

### Viewing Lead Score
1. Open lead details
2. View score in the header
3. Hover to see score breakdown

### Customizing Scoring
1. Go to Settings > Lead Scoring
2. Adjust point values
3. Add custom rules
4. Save changes

## Lead Sources

### Default Sources
- Website
- Phone
- Email
- Referral
- Social Media
- Open House
- Walk-in

### Adding Custom Sources
1. Go to Settings > Lead Sources
2. Click "Add New Source"
3. Enter source name
4. Set as active/inactive

## Lead Tags

### Adding Tags
1. Open lead details
2. Click "Add Tag"
3. Select existing or create new
4. Click "Save"

### Managing Tags
1. Go to Settings > Tags
2. Create, edit, or delete tags
3. Set tag colors

## Lead Reports

### Available Reports
- Lead Source Report
- Conversion Report
- Agent Performance
- Status Distribution
- Lead Response Time

### Generating Reports
1. Go to Reports > Leads
2. Select report type
3. Set date range
4. Apply filters
5. Click "Generate"
6. Export as needed (PDF, Excel, CSV)

## Lead Imports

### Preparing Your File
1. Download the template
2. Fill in required fields:
   - First Name
   - Last Name
   - Email/Phone
   - Source
   - Status

### Importing Leads
1. Go to Leads > Import
2. Upload your file
3. Map the fields
4. Preview the import
5. Click "Import Leads"

## Lead Exports

### Exporting Leads
1. Apply any filters
2. Click "Export"
3. Select format (CSV, Excel)
4. Choose fields to include
5. Click "Export"

- Look in trash/deleted

- Use the merge tool

#### Notification Issues
- Check notification settings
