# प्रोजेक्ट रिऑर्गनाइजेशन प्लान

## वर्तमान स्थिति

वर्तमान में प्रोजेक्ट में निम्नलिखित समस्याएं हैं:

1. कई डुप्लिकेट फाइल्स (जैसे `updated-*.php`, `*_backup.php`, `edit-profile - Copy.php` आदि)
2. डेटाबेस फाइल्स अलग-अलग स्थानों पर हैं (`DATABASE FILE` फोल्डर और रूट डायरेक्टरी में)
3. CSS फाइल्स दो अलग-अलग स्थानों पर हैं (`assets/css` और `css` फोल्डर)
4. JS फाइल्स दो अलग-अलग स्थानों पर हैं (`assets/js` और `js` फोल्डर)
5. PHP इंक्लूड फाइल्स दो अलग-अलग स्थानों पर हैं (`include` और `includes` फोल्डर)

## प्रस्तावित संरचना

### 1. एसेट्स का संगठन

- **assets/** - सभी स्टैटिक एसेट्स के लिए
  - **css/** - सभी CSS फाइल्स
  - **js/** - सभी JavaScript फाइल्स
  - **images/** - सभी इमेज फाइल्स
  - **fonts/** - सभी फॉन्ट फाइल्स
  - **vendor/** - थर्ड-पार्टी लाइब्रेरीज

### 2. PHP इंक्लूड्स का संगठन

- **includes/** - सभी PHP इंक्लूड फाइल्स
  - **config/** - कॉन्फिगरेशन फाइल्स
  - **classes/** - क्लास फाइल्स
  - **functions/** - फंक्शन फाइल्स
  - **templates/** - टेम्पलेट फाइल्स

### 3. डेटाबेस फाइल्स का संगठन

- **database/** - सभी डेटाबेस फाइल्स
  - **migrations/** - माइग्रेशन फाइल्स
  - **backups/** - बैकअप फाइल्स
  - **scripts/** - डेटाबेस स्क्रिप्ट्स

### 4. अपलोड्स का संगठन

- **uploads/** - सभी अपलोड्स
  - **property/** - प्रॉपर्टी इमेजेज
  - **users/** - यूजर इमेजेज

### 5. डॉक्युमेंटेशन का संगठन

- **docs/** - सभी डॉक्युमेंटेशन फाइल्स

## कार्यान्वयन योजना

### चरण 1: डुप्लिकेट फाइल्स की समीक्षा और हटाना

1. `updated-*.php` फाइल्स की समीक्षा करें और उन्हें मूल फाइल्स से मर्ज करें या हटा दें
2. `*_backup.php` फाइल्स की समीक्षा करें और उन्हें हटा दें
3. अन्य डुप्लिकेट फाइल्स की समीक्षा करें और उन्हें हटा दें

### चरण 2: फोल्डर संरचना का निर्माण

1. `database` फोल्डर बनाएं और सभी डेटाबेस फाइल्स को उसमें मूव करें
2. `includes` फोल्डर में सभी PHP इंक्लूड फाइल्स को मर्ज करें
3. `assets/css` में सभी CSS फाइल्स को मर्ज करें
4. `assets/js` में सभी JS फाइल्स को मर्ज करें

### चरण 3: फाइल्स को नई संरचना में मूव करना

1. सभी डेटाबेस फाइल्स को `database` फोल्डर में मूव करें
2. सभी PHP इंक्लूड फाइल्स को `includes` फोल्डर में मूव करें
3. सभी CSS फाइल्स को `assets/css` फोल्डर में मूव करें
4. सभी JS फाइल्स को `assets/js` फोल्डर में मूव करें

### चरण 4: पाथ अपडेट करना

1. सभी PHP फाइल्स में इंक्लूड पाथ अपडेट करें
2. सभी HTML/PHP फाइल्स में CSS और JS पाथ अपडेट करें

## सावधानियां

1. किसी भी फाइल को हटाने से पहले बैकअप लें
2. एक-एक करके परिवर्तन करें और हर परिवर्तन के बाद टेस्ट करें
3. पाथ अपडेट करते समय सावधानी बरतें
