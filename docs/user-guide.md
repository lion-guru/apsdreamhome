## 🚀 Getting Started

### First Time Users
1. **Visit the Website**: Go to [https://yourdomain.com](https://yourdomain.com)
2. **Register Account**: Click "Sign Up" and create your account
3. **Verify Email**: Check your email for verification link
4. **Complete Profile**: Add personal details and preferences
5. **Start Exploring**: Browse properties or list your own

### Account Types
- **Customer**: Browse and inquire about properties
- **Agent**: List and manage properties, handle leads
- **Associate**: MLM network member, earn commissions
- **Employee**: Administrative access
- **Admin**: Full system administration

## 👤 User Registration & Login

### Creating an Account
1. Click **"Sign Up"** on the homepage
2. Fill in your details:
   - Full Name
   - Email Address
   - Phone Number
   - Password (minimum 8 characters)
   - Account Type (Customer/Agent/Associate)
3. Agree to Terms & Conditions
4. Click **"Create Account"**
5. Verify your email address

### Logging In
1. Click **"Login"** in the top navigation
2. Enter your email and password
3. Click **"Sign In"**
4. Use **"Remember Me"** for convenience

### Password Recovery
1. Click **"Forgot Password?"** on login page
2. Enter your registered email
3. Check email for reset instructions
4. Follow the link to create new password

## 🏠 Property Search & Browsing

### Basic Property Search
1. Use the **search bar** on homepage
2. Enter location, property type, or keywords
3. Click **"Search"** or press Enter
4. Browse results with filters and sorting

### Advanced Filters
- **Property Type**: Apartment, Villa, Commercial, Plot
- **Price Range**: Minimum and maximum budget
- **Location**: City, area, or pincode
- **Bedrooms**: 1BHK, 2BHK, 3BHK, etc.
- **Status**: Available, Sold, Under Construction
- **Features**: Furnished, Parking, Garden, etc.

### Sorting Options
- **Newest First**: Recently added properties
- **Price: Low to High**: Budget-friendly first
- **Price: High to Low**: Premium properties first
- **Featured**: Promoted listings

### Saved Searches
1. Set up filters for your preferred properties
2. Click **"Save Search"**
3. Get email notifications for new matches
4. Manage saved searches in your dashboard

## 📋 Property Details & Enquiries

### Viewing Property Details
1. Click on any property card from search results
2. View comprehensive details:
   - High-quality images and virtual tours
   - Complete specifications (area, rooms, amenities)
   - Location map and nearby facilities
   - Price and payment options
   - Agent contact information

### Making an Enquiry
1. Click **"Contact Agent"** or **"Send Enquiry"**
2. Fill in your details (if not logged in)
3. Write your message or questions
4. Upload documents if needed
5. Click **"Send Enquiry"**
6. Receive confirmation and agent response

### Requesting Site Visit
1. On property details page, click **"Schedule Visit"**
2. Choose preferred date and time
3. Select visit type (individual or group)
4. Add special requirements
5. Confirm booking
6. Receive confirmation and reminders

## 👨‍💼 Agent Dashboard

### Getting Started as an Agent
1. Register as an Agent during sign-up
2. Complete your agent profile
3. Upload necessary documents (license, ID)
4. Wait for admin approval
5. Access your agent dashboard

### Managing Properties
#### Adding New Properties
1. Go to **"My Properties"** → **"Add Property"**
2. Fill in property details:
   - Basic information (title, description, price)
   - Property specifications (area, rooms, amenities)
   - Location details (address, coordinates)
   - Media uploads (photos, videos, virtual tours)
3. Set visibility (public/private)
4. Click **"Publish Property"**

#### Editing Properties
1. Go to **"My Properties"** in dashboard
2. Click **"Edit"** on any property
3. Make necessary changes
4. Save and update

#### Property Analytics
- View property views and enquiries
- Track interested buyers
- Monitor conversion rates
- Generate performance reports

### Lead Management
1. Access **"My Leads"** in dashboard
2. View all enquiries and interested buyers
3. Update lead status (New, Contacted, Qualified, Won, Lost)
4. Schedule follow-ups and site visits
5. Convert leads to successful deals

### Commission Tracking
- View earned commissions
- Track pending payments
- Generate commission reports
- Monitor performance metrics

## 🤝 Associate Portal

### Network Management

### Commission Dashboard

- **Payment History**: View all received payments

- **Network Statistics**: Monitor your team's performance

### Marketing Tools

## 💳 Payment & Transactions

### Making Payments
1. Choose a property and contact the agent
2. Discuss terms and finalize agreement
3. Initiate payment through secure gateway
4. Select payment method (Card, UPI, Net Banking)
5. Complete transaction with OTP verification

### Payment Methods Supported
- **Credit/Debit Cards**: Visa, Mastercard, RuPay
- **UPI**: Google Pay, PhonePe, Paytm, BHIM UPI
- **Net Banking**: All major Indian banks
- **Wallets**: Paytm, Mobikwik, Ola Money
- **EMI Options**: Available for high-value transactions

### Transaction Security
- **SSL Encryption**: 256-bit SSL protection
- **PCI Compliance**: Secure payment processing
- **OTP Verification**: Two-factor authentication
- **Transaction Monitoring**: Fraud detection systems

### Payment History
- View all your transactions
- Download payment receipts
- Track refund status
- Contact support for issues
