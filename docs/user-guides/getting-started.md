# Getting Started with APS Dream Home

Welcome to APS Dream Home! This guide will help you get started with the system.

## System Requirements
- Web browser (Chrome, Firefox, Safari, or Edge)
- Internet connection
- Screen resolution: 1280x768 or higher

## Logging In

1. Open your web browser and navigate to your APS Dream Home URL
2. Enter your username and password
3. Click the "Login" button

![Login Screen](/assets/images/login-screen.png)

## Dashboard Overview

After logging in, you'll see your dashboard which provides an overview of:
- Upcoming appointments
- Recent activities
- Important notifications
- Quick access to common tasks

## Navigation

The main menu is located on the left side of the screen:
- **Dashboard** - Your main workspace
- **Properties** - Manage property listings
- **Customers** - Customer database
- **Bookings** - Manage property bookings
- **Finance** - Financial transactions and reports
- **Reports** - Generate and view reports
- **Settings** - System configuration

## User Profile

To update your profile:
1. Click on your name in the top-right corner
2. Select "My Profile"
3. Update your information
4. Click "Save Changes"

## Changing Your Password

1. Go to "My Profile"
2. Click on "Change Password"
3. Enter your current password
4. Enter your new password
5. Confirm the new password
6. Click "Update Password"

## Getting Help

If you need assistance:
1. Check the help documentation (click the "?" icon)
2. Contact your system administrator
3. Email support@apsdreamhome.com
