# Admin Guide

This guide provides detailed information for administrators managing the APS Dream Home system.

## User Management

### Creating Users
1. Navigate to Admin Panel > Users > Add New
2. Fill in user details
3. Assign appropriate role (Admin, Agent, User)
4. Set initial password or enable password reset

### Managing Permissions
1. Go to Admin Panel > Roles & Permissions
2. Create or edit roles
3. Set granular permissions for each role

## Property Management

### Adding Properties
1. Go to Properties > Add New
2. Fill in property details
3. Upload high-quality images
4. Set pricing and availability

### Property Categories
1. Navigate to Properties > Categories
2. Add or edit property categories
3. Set category-specific settings

## System Configuration

### General Settings
- Site title and description
- Contact information
- Currency and date formats

### Email Configuration
1. Go to Settings > Email
2. Configure SMTP settings
3. Test email functionality

## Security

### User Authentication
- Enable/disable registration
- Set password policies
- Configure two-factor authentication

### Data Backup
1. Go to System > Backup
2. Create manual backups
3. Configure automatic backup schedule

## Reporting

### Generating Reports
1. Navigate to Reports
2. Select report type
3. Set date range and filters
4. Export to PDF/Excel

## Troubleshooting

### Common Issues
- **Login Issues**: Reset password or check user status
- **Email Not Sending**: Verify SMTP settings
- **Slow Performance**: Check server resources and optimize database
